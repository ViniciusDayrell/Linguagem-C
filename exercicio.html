Projeto: Sistema de Cadastro de Usuário (C# Console)

Desenvolver um programa em C# que leia dados do usuário pelo console e armazene essas informações em um arquivo de texto (.txt).
O objetivo é praticar entrada e saída de dados, uso de variáveis e gravação de arquivos.

Descrição

Crie um programa console em C# que:

  • ≫Exiba uma mensagem de boas-vindas
  • ≫Solicite os seguintes dados do usuário: nome, idade, e-mail
  • ≫Guarde os dados em variáveis correspondentes.
  • ≫Mostre os dados digitados no console, confirmando a entrada.
  • ≫Grave essas informações em um arquivo de texto chamado usuarios.txt.
