Estruturas Condicionais

As estruturas condicionais em C controlam o fluxo de um programa com base em condições lógicas. As principais são if, if-else e switch-case, que permitem a execução de diferentes blocos de código dependendo se uma expressão é verdadeira ou falsa.

IF

  • Executa um bloco de código apenas se a condição for verdadeira.
  • Sintaxe: 
    
if (condição) { 
    // bloco de código 
}
  • Exemplo: 
    #include <stdio.h>

int main() {
    int idade;
    printf("Digite sua idade: ");
    scanf("%d", &idade);

    if (idade >= 18) {
        printf("Você é maior de idade.\n");
    }
    return 0;
}

IF-ELSE

  • Executa um bloco de código se a condição for verdadeira e outro bloco se for falsa.
  • Sintaxe: 
    
if (condição) {
    // código se verdadeiro 
} else {
    // código se falso
}
  • Exemplo: 
    
int x = 10;
if (x > 5) {
    printf("Maior que 5");
} else {
    printf("Menor ou igual a 5");
}

SWITCH-CASE

  • Permite escolher uma entre várias opções de código com base no valor de uma variável.
  • A expressão do switch deve ser comparada com constantes em cada bloco case
  • A palavra-chave break é essencial para sair do switch após a execução de um bloco.
  • A cláusula default é executada se o valor da variável não corresponder a nenhuma das opções de case.
  • Sintaxe: 
    
switch (variavel) { 
    case constante1: 
        // código 
        break;
    case constante2: 
        // código 
        break;
    default: 
        // código 
        break;
}
  • Exemplo: 
    
#include <stdio.h>

int main() {
    int opcao;
    printf("=== MENU DE OPÇÕES ===\n");
    printf("1 - Café\n");
    printf("2 - Suco\n");
    printf("3 - Água\n");
    printf("4 - Refrigerante\n");
    printf("Escolha uma opção: ");
    scanf("%d", &opcao);
    
    switch (opcao) {
        case 1: printf("Você escolheu Café!\n"); break;
        case 2: printf("Você escolheu Suco!\n"); break;
        case 3: printf("Você escolheu Água!\n"); break;
        case 4: printf("Você escolheu Refrigerante!\n"); break;
        default: printf("Opção inválida!\n");
    }
    return 0;
}

Estrutura Condicional Simples

A estrutura condicional simples expõe que uma instrução ou bloco de instruções só sejam executados se determinada condição for verdadeira. Uma condição é uma comparação que possui dois valores possíveis: verdadeiro ou falso.

Estrutura Condicional Composta

A estrutura condicional composta expõe que uma instrução ou bloco de instruções só sejam executados se determinada condição for satisfeita. Caso essa condição seja falsa, é executada uma outra instrução ou um bloco de instruções diferente.

Exercícios Práticos

Exercício 1 - Cálculo das raízes de uma equação do 2º grau

Crie um programa em C que leia os valores de a, b e c de uma equação do 2º grau e determine suas raízes.

Se Δ < 0 → não existem raízes reais.
Se Δ = 0 → existe apenas uma raiz real.
Se Δ > 0 → existem duas raízes reais diferentes. 


#include <stdio.h>
#include <math.h>

int main() {
    float a, b, c, delta, x1, x2;
    printf("Digite os valores de a, b e c: ");
    scanf("%f %f %f", &a, &b, &c);

    if (a == 0) {
        printf("Não é uma equação do segundo grau.\n");
    } else {
        delta = b*b - 4*a*c;
        if (delta < 0) {
            printf("Não existem raízes reais.\n");
        } else if (delta == 0) {
            x1 = -b / (2*a);
            printf("Existe apenas uma raiz real: %.2f\n", x1);
        } else {
            x1 = (-b + sqrt(delta)) / (2*a);
            x2 = (-b - sqrt(delta)) / (2*a);
            printf("As duas raízes reais são: %.2f e %.2f\n", x1, x2);
        }
    }
    return 0;
}

Exercício 2 - Calculadora com Switch-Case

Crie um programa em C que funcione como uma calculadora simples.

  • Ler dois números do usuário.
  • Ler a operação desejada (+, -, *, /).
  • Usar switch-case para realizar a operação correta.
  • Exibir o resultado formatado.
  • Tratar o caso de divisão por zero.
  • Exibir mensagem de erro para operações inválidas.

#include <stdio.h>

int main() {
    float num1, num2, resultado;
    char operacao;
    
    printf("=== CALCULADORA SIMPLES ===\n");
    printf("Digite o primeiro número: ");
    scanf("%f", &num1);
    printf("Digite a operação (+, -, *, /): ");
    scanf(" %c", &operacao);
    printf("Digite o segundo número: ");
    scanf("%f", &num2);
    
    switch (operacao) {
        case '+':
            resultado = num1 + num2;
            printf("%.2f + %.2f = %.2f\n", num1, num2, resultado);
            break;
        case '-':
            resultado = num1 - num2;
            printf("%.2f - %.2f = %.2f\n", num1, num2, resultado);
            break;
        case '*':
            resultado = num1 * num2;
            printf("%.2f * %.2f = %.2f\n", num1, num2, resultado);
            break;
        case '/':
            if (num2 != 0) {
                resultado = num1 / num2;
                printf("%.2f / %.2f = %.2f\n", num1, num2, resultado);
            } else {
                printf("Erro: Divisão por zero!\n");
            }
            break;
        default:
            printf("Operação inválida! Use apenas +, -, * ou /\n");
    }
    return 0;
}
