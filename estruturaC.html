Estrutura de um programa em C

Todo programa em linguagem C segue uma estrutura básica, que serve como o esqueleto do código. É a partir dessa base que construímos qualquer programa, do mais simples ao mais complexo.

Bibliotecas

  • A primeira parte de um programa em C é a inclusão das bibliotecas, que fornecem funções prontas para o uso.
  • Essas bibliotecas são incluídas com a diretiva #include.
  • Exemplo: 
    
#include <stdio.h>   // Biblioteca para entrada e saída de dados (printf, scanf)
#include <stdlib.h>  // Biblioteca auxiliar com funções úteis (ex: system, malloc)

Função Principal

Todo programa em C precisa ter uma função principal, chamada main(). É dentro dela que o computador começa a executar os comandos.


Exemplo:


int main() {
    // Aqui dentro vão os comandos do programa
    return 0;
}
  • A palavra int indica que a função vai retornar um valor inteiro para o sistema operacional ao final da execução.
  • O corpo da função fica entre chaves { }, onde você escreve o código que o programa deve executar.

Retorno do Programa (return 0;)

O comando: return 0; significa que o programa foi executado com sucesso.


Como o main foi declarado como int, ele precisa retornar um número inteiro, e por convenção usamos o zero (0) para indicar que tudo ocorreu bem.


Exemplo completo

                    
#include <stdio.h>
#include <stdlib.h>
                        
int main() {
    printf("Ola, mundo!\n"); // Exibe uma mensagem na tela
                            
    return 0; // Encerra o programa com sucesso
}

Exercicios Práticos

Monte um programa completo em C que:

  • Contenha as bibliotecas básicas (stdio.h e stdlib.h)
  • Tenha a função principal int main()
  • Mostre na tela a mensagem: Ola, mundo! Meu primeiro programa em C!

#include <stdio.h>
#include <stdlib.h>

int main() {
    printf("Ola, mundo! Meu primeiro programa em C!\n");

    return 0;
}
